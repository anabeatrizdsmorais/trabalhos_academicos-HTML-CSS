Exercicios JavaScript - Programação WEB

Desenvolver os seguintes algorítmos utilizando os conceitos de javascript vistos até o momento. Ao final, zipar a pasta com todos os exercícios e postar a resolução dos mesmos.

1) Desenvolver um algorítimo que solicite ao usuário 3 números, calcule a média dos mesmos e informe o resultado ao usuário final. (Média de 3 números é (n1 + n2 + n3) / 3).

2) Desenvolva um algorítimo que solicite ao usuário 3 números, e a final, informe qual dos números é o maior ou se os números são iguais.

3) Desenvolva um algorítimos que solicite ao usuário 2 números, e ao final, informe a divisão do primeiro pelo segundo.

4) Desenvolva um algorítimos que solicite ao usuário 2 números, e ao final, informe a multiplicação do primeiro pelo segundo.
